# DeveloperHub

Mini fullstack project for learning backend development with Java and Spring Boot.

## Tech Stack

- Java 21
- Spring Boot
- React
- Vite
- JUnit 5
- Mockito

## Features

- REST API
- DTO architecture
- validation
- global exception handling
- unit testing
- duplicate email protection
- custom exceptions
